# ClickSHOP 🛍️

**Modern E-commerce Website** – Built with pure HTML & CSS

Live Demo: [https://raneaaditi1-byte.github.io/ClickSHOP/](https://raneaaditi1-byte.github.io/ClickSHOP/)

---

## ✨ Features

- Fully responsive design (mobile + desktop)
- Clean and modern UI/UX
- Featured products section (Fashion – Men, Women & Kids)
- Promotional banner with discount code
- Simple and elegant product cards
- Contact & feedback section
- No backend required (pure static site)

## 🎯 Purpose

This is a **static front-end demonstration** of an e-commerce website focused on trendy fashion and footwear. It showcases clean design, smooth layout, and responsive behavior.

## 🛠️ Technologies Used

- **HTML5**
- **CSS3** (including custom styles and UI components)
- Responsive design with Flexbox/Grid
- No JavaScript frameworks (vanilla)
